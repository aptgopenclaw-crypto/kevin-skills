## 五、注意事項

1. **日期格式**：網站使用民國年（如 `114/03/04`），Excel 輸出保留原格式。  
2. **驗證碼**：列表頁（`readTenderAgent`）可能需通過撲克牌 CAPTCHA；  
   詳細頁（`QueryAtmAwardDetail?pkAtmMain=...`）為 `WithoutSso` 公開頁，通常無 CAPTCHA。  
3. **多家得標廠商**：複數決標時，Excel 僅列第一家得標廠商；完整廠商資料儲存於 `_bidders` 欄位。  
4. **難字顯示**：廠商名稱可能含難字圖片，解析時以純文字取代。  
5. **頁面切換**：若搜尋結果超過 50 筆需翻頁；程式自動處理換頁（`d-16396-p=N`）。
